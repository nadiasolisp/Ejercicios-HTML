Términos de javascript

Función
Es un procedimiento, un conjunto de instrucciones que realizan una tarea específica o calculan un valor.
let
Declara una variable limitando el alcance al bloque donde se está usando. Es reasignable.
const
Declara una constante que tiene alcance global o local para el bloque donde se declara, no puede redeclararse.
Etiqueta < a >
Crea un enlace a una página web. Trabaja con el atributo href, el cual es donde se coloca el enlace.
Etiqueta < mark >
Se utiliza para resaltar texto en el documento.
Etiqueta < audio >
Se utiliza para insertar contenido de audio.
